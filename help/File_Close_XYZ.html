Closing Input Files

Use the Close X, Y, or forecast option under the File menu item to close an input file without restoring CPT's default program settings (as occura with File ~ New). Closing an X or Y file will reset CPT so that results need to be recalculated. If a forecast file is closed only the forecasts will be reset.
