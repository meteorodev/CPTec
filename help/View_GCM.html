Raw GCM Output (GCM)

From the Introductory Window the Raw GCM Output (GCM) menu item will open the CPT Window for GCM. This menu item is not available if a station or unreferenced X input file is open: GCM is only available using gridded X files. This menu item is not available if an unreferenced Y input file is open: GCM is only available using station or gridded Y files. Similarly, the MLR menu item is not available if a probabilistic X file is open.

The GCM option compares GCM model output with station or gridded data on a gridbox-by-gridbox basis. For each station or gridbox in the Y file the nearest GCM grid, or interpolated GCM values from the 4 nearest gridboxes (depending upon the option set in Options ~ GCM Options) is compared. This procedure is in contrast to the CCA and PCR options, which attempt to perform some form of spatial correction to the model output. (The MLR option does not permit the use of gridded X files.) If the X input file contains multiple fields the forecast combination option indicated in Options ~ GCM Options) will be used.
