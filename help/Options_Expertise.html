Expertise Level

The Expertise Level menu permits the user to switch between a Beginner and an Expert user level. The level that is activated is greyed out; for example, if expert level is selected then only the beginner option will appear.

The expertise level affects the menu options shown under View as well as the availability of some results. If a Beginner level is used, the View menu items change to:

  • Forecast, which runs Canonical Correlation Analysis (CCA) with the Synchronous Predictors option switched off.
  • Diagnostics, which runs Canonical Correlation Analysis (CCA) with the Synchronous Predictors option switched on.
  • Verification), which runs Probabilsitic Forecast Verification (PFV)
The Beginner level simplifies the Tools ~ Validation ~ Performance window by removing the ROC diagram and showing only the most useful scores.
