Saving the Results

Most of the results can be saved in files by using the Output Results option under the File menu item. This option becomes accessible only after the calculations are complete. A variety of results can be saved:

  • cross-validated predictions;
  • retroactive predictions;
  • X input data;
  • Y input data.
  • X and Y eigenvalues;
  • X and Y spatial loadings;
  • X and Y temporal scores;
  • X and Y homogeneous covariance maps and map series;
  • X and Y homogeneous scores;
  • canonical correlations;
  • regression coefficients;
  • forecasts.
The input files are saved with any missing values replaced. Skill maps and verification results can be saved also, but only by right-clicking on the skill map window, or a verification window. They cannot be saved from the standard output files window because the skill scores and the verification results are stored in temporary memory. Forecasts (including principal component scores for new values of the predictors that are not part of the training period) can be saved only once a forecast has been viewed using one of the Tools ~ Forecast menu items.

File names for each of the desired outputs need to be specified. By default these files are stored in the OUTPUT subdirectory. The formats follow standard FORTRAN formatting conventions, except that formatted direct access is not permitted. These conventions are not discussed in these Help Pages. It is recommended that the default settings of formatted sequential output be used to obtain standard text (ASCII) files in which fields are separated by tabs. Formatted output files are given the .txt file extension, and are saved in CPT version 10 format, while unformatted files are given the .dat extension.
