Sort Ensemble Members

If the predictor input data consist of individual ensemble members, these will be input as separate fields. In most cases it would make sense to sort theensemble members so that the first ensemble member always has the lowest value of all the members, and the last ensemble members is the highest value. If the members are sorted then the first ensemble member will therefore represent the driest / coolest forecast of the members, and the last ensemble member will be the wettest / hottest forecast.

Selecting the Sort Ensemble Members menu item will toggle the sorting on and off. A tick mark is shown by the menu item if the option is on.
